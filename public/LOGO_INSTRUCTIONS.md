# Logo Image Instructions

## How to Add Your Logo

1. **Place your golden statue image file** in this `public` folder
2. **Name it exactly:** `logo-statue.png`
3. **Recommended format:** PNG with transparent background
4. **Recommended size:** At least 512x512 pixels (larger is better)

## File Location

```
public/
  └── logo-statue.png  ← Place your image here
```

## Image Requirements

- **Format:** PNG (preferred) or JPG
- **Background:** Transparent (PNG) or light/neutral color
- **Size:** Minimum 512x512px, ideally 1024x1024px or larger
- **Content:** Golden statue image as described

The logo will automatically:
- Display with elegant gold glow effects
- Blend seamlessly with the black & gold theme
- Scale responsively on all devices
- Show a fallback "GCMA" text if image is missing

## After Adding the Image

1. Save the image file in this folder
2. Refresh your browser
3. The logo should appear in the Hero section
